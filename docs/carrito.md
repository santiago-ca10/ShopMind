# 🛒 Carrito de compras

El carrito funciona completamente en frontend.

---

## Persistencia

Se almacena usando:

- localStorage
- Context API

---

## Funciones

- agregar productos
- eliminar productos
- aumentar cantidad
- disminuir cantidad
- limpiar carrito

---

## Nota

>   El carrito NO se almacena en MongoDB.
